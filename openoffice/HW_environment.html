Description of the OpenOffice To OpenVMS porting hardware.

The following hardware is currently used by the porting team:

Alphaserver 1200, 1 CPU, 1 GB memory (openVMS 7.3-2)
storage: 1 x CDROM, 6 x 9.1 GB disk, 4 x 4.3 GB disk

Sun sparc station 5, 128 MB memory (Solaris 9)
storage: 1 x CDROM, 9.1 GB disk

UNISYS Aquanta DM PC, pentium II 233 MHz, 128 MB (REDHAT linux 8.0)
storage: 1 x CDROM, 2 x 4.3 GB disk
